La pseudo-clase :nth-child() de CSS coincide con uno o más elementos en función de su posición entre un grupo de hermanos. Para ello, el numero que pongamos entre los parentesis es al "hijo" al que estas haciendo referencia, si ponemos el 1 sera el primero listado en el html, 2 el segundo, y asi sucesivamente. En el caso de poner "n", nos estaremos refiriendo a todos los elementos de clase "hijo"

CSS DEMO :nth-child()

.hijo:nth-child() {

}
