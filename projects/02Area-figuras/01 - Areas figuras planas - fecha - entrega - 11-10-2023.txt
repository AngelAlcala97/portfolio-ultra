Realizar una (única) página que calcule las áreas de las siguientes figuras:

Rectángulo
Triángulo
Rombo
Cuadrado
Paralelogramo
Trapecio
Romboide
Polígono regular
Círculo

Para las fórmulas se puede tomar la siguiente página web como referencia:

https://www.monografias.com/trabajos88/deduccion-formulas-calcular-area-figuras-planas/deduccion-formulas-calcular-area-figuras-planas.shtml

La página tendrá una sección (hecha con anclas) para cada figura, con cajas de texto para introducir los datos y un botón para realizar el cálculo. Cuando se obtenga el resultado se mostrará en un contenedor en esa sección. Se añadirá también una imagen de cada figura que puede tomarse de la página propuesta para su resolución.

La página deberá incluir un índice con secciones para ir al apartado de cada figura. Asimismo en cada apartado deberá aparecer un enlace para volver al índice.

Versión 2:

Tener una versión alternativa en la que disponemos de un botón para cada figura en lugar de un enlace.
Al pulsar el botón aparecerá el código de la figura (el html que la muestra) con su botón correspondiente para llamar a la función de cálculo y su contenedor correspondiente para mostrar el resultado.

Se recompensa con 0,1 por entregar esta versión adicional (hay que hacer la versión 1 también)

Versión IA:

Se recompensa con 0,05 por entregar una versión adicional generada con IA tanto código como imágenes (Guardar los prompts para comparar)
